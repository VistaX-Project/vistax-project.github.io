VistaX Project Homepage

The VistaX Project aims to make unofficial installers of unsupported software for Windows Vista that can be used with the Extended Kernel.

This helps the user easily install unsupported software on Vista but also makes the software more integrated with the OS.

Status

As of now, there are no installers available since the project is still in beta.
